================= CSS - Naming ==================
=================================================
- BEM naming rule로 작성
- 파일 이름__태그역할 의 구조로 작성
- 태그 div로 감싸 들어갈 때 section -> container -> part -> ele(element) -> box -> piece 순으로 나열할 것
- 나누는 기호는 - 로 작성 (ex. main-app__container-part-ele-text)
